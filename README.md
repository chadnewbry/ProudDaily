=======

An LGBTQ+ daily affirmation iOS app built with SwiftUI.

## Features

- **Daily Affirmations** — A new affirmation each day to start your morning right
- **Library** — Browse affirmations by category (Self Love, Identity, Community, Resilience, Joy, Relationships)
- **Journal** — Personal writing space to reflect on your affirmations
- **Progress** — Track your streaks and engagement stats
- **Widgets** — Home screen and lock screen widgets via WidgetKit
- **Pride Themes** — Color themes for rainbow, trans, bi, pan, nonbinary, lesbian, and ace pride flags

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+

## Tech Stack

- SwiftUI
- WidgetKit
- Observation framework (`@Observable`)

## Getting Started

1. Clone the repo
2. Open `Proud Daily.xcodeproj` in Xcode
3. Build and run on simulator or device
